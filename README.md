# 🛡️ Cybersecurity Checklist

> A practical guide to protect websites and digital assets from common cyber threats.

---

## 🚨 Why This Matters

Most websites don’t get hacked because of advanced attacks…  
They get hacked because of simple mistakes.

This checklist helps you eliminate those mistakes.

---

## 🔐 Essential Security Checklist

### 🔑 Authentication
- Use strong and unique passwords
- Enable Two-Factor Authentication (2FA)
- Avoid password reuse

### 🧱 Website Security
- Keep all plugins and systems updated
- Remove unused plugins/themes
- Use secure hosting

### 🌐 Network Protection
- Use HTTPS (SSL certificate)
- Protect against DDoS attacks
- Use a firewall (WAF)

### 🧠 Human Awareness
- Don’t click unknown links
- Avoid downloading suspicious files
- Educate team members

---

## ⚠️ Common Mistakes

- Weak passwords
- Outdated software
- No backups
- Misconfigured servers

---

## 🚀 Goal

Stay secure. Stay aware.  
Because in cybersecurity… you're either protected or targeted.
